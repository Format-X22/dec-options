## Description

Options aggregator with api.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# build all
$ npm run build:all

# api microservice (dev)
$ npm run start:api

# aggregator microservice (dev)
$ npm run start:agg

# production bare metal mode
$ npm run start:prod:api
$ npm run start:prod:agg
```
